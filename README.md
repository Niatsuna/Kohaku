# Kohaku
Rust &amp; Python based project for collecting game information &amp; providing a (sassy) interaction via discord.

# Disclaimer:
Kohaku started as a hobby discord bot project to learn python at the time. Over the years this child slowly was forgotten. It is now time to revive it.
Plan currently is to collect game data of games friends and I play, format them, analyze them and provide them through a discord integration.

This code will not be 100% pretty.
